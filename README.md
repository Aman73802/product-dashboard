# 🛒 Product Dashboard

A modern, responsive **Product Dashboard** built using **React.js**, **Tailwind CSS**, and **ShadCN/UI**. This project allows users to add products, view them in a cart, and perform basic e-commerce operations in a clean and minimal interface.


## 🚀 Features

- 🛒 Add to Cart Functionality
- ❌ Remove from Cart Option
- 💵 Checkout Option with Total Calculation
- 📱 Fully Responsive Design

---

## ⚙️ Tech Stack

- **React.js** – Frontend library
- **Tailwind CSS** – Utility-first CSS framework
- **ShadCN/UI** – Beautiful pre-built components
- **React Router DOM** – For routing pages
- **Vite** – Fast development server (if used)

🛠️ Getting Started
1. Clone the Repository
git clone https://github.com/Aman73802/product-dashboard.git
cd product-dashboard


2. Install Dependencies
npm install

3. Start the Development Server
npm run dev
Then open http://localhost:5173 in your browser.

📦 Build for Production
npm run
